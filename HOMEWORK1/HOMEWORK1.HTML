Vocab Definitions

  1. HTTP
    • Definition: It is the used the secure lanuage used by computers to communicate with each other over the internet
    • Official MDN Web Doc
  2. HTML
    • Definition:Descriptive language use to design websites and sets its structure
    • Official MDN Web Doc
  3. ELEMENT
    • Definition:It is a part of the webpage that can contain a piece of data, text, or a picture. it typically includes an opening tag, a text content, and a closing statement.
    • Official MDN Web Doc
  4. TAG
    • Definition:It is used for creating an element. It typically contains an opening bracket and a closing bracket.
    • Official MDN Web Doc
  5. Block (CSS)
    • Definition:It is an HTML element that appears in a new line an example of this is the paragraph element < p>.
    • Official MDN Web Doc
  6. Doctype
    • Definition:it is something that is needed at the beginning of all documents. It's only purpose is to make it so that the webpage will not change any relevant specifications pertaining to a document.
    • Official MDN Web Doc
  7. Head
    • Definition:it's the part of the HTML document that contains metadata about the document. Data like version number, author, descriptions etc.
    • Official MDN Web Doc
  8. Attribute
    • Definition:It extends a tag and adds meta data. It always has a form of name=value.
    • Official MDN Web Doc
  9. Boolean
    • Definition:It is a data stype that can either be true or false.
    • Official MDN Web Doc
  10. Dom
    • Definition:It is an appplication programming interface (API) for any HTML or XML document. It allows for accesing any node within a document
    • Official MDN Web Doc
  11. CSS
    • Definition:It's a declarative language that shows how the webpage would look in the browser.
    • Official MDN Web Doc
  12. Property (CSS)
    • Definition:It's a characteristic whose value defines as aspect of how the browser would display the element.
    • Official MDN Web Doc
  13. Selector (CSS)
    • Definition:It's the part of the document that says which element in a document the rule will match.
    • Official MDN Web Doc
